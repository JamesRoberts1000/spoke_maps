Visualising connections to the nearest foodbanks

Welcome to the UK foodbank network mapping tool

This map visualizes the UK's foodbank network using a hub and spoke pattern. Here's how to use it:

  • The map center (hub) connects to the 8 nearest foodbanks
  • Pan the map to see connections update dynamically
  • Hover over points to see foodbank names
  • Click points for detailed information
